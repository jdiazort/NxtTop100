    Sofware Platform

    Back End: Python
    . Data Collection
    . Data Analysis
    . Regression Algorithm
    Front End: Javascript
    . Charting visualization: amCharts
    . Angular Framework
    Database: MySQL

    Data Gathering

    Data will be collected from the web. We will use an unofficial python API for accessing charts from Bilboard.com. https://github.com/guoguo12/billboard-charts

    Labor Division

    All group members will contribute to all parts of the project. Still, major responsibilities were assigned to each group member.
    Antonio Minondo - Data Collection and Storage
    Isiah Leblanc - Front End
    Jorge Diaz - Data visualization and trend charts
    The most crucial piece of our project is the algorithm that will predict which position a song will reach the Top100 in a given future time. We are going to work on it as a group.
